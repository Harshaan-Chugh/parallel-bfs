# Test Graphs for Parallel BFS

## Format

All graphs use a simple **edge list** text format (`.edgelist`):

```
<num_vertices> <num_edges>
<src> <dst>
<src> <dst>
...
```

- Vertices are **0-indexed**.
- Edges are **undirected** — each edge is listed once (e.g., `0 1` means an edge between 0 and 1 in both directions). Your loader should add both directions.
- Lines starting with `#` are comments and should be skipped.
- The first non-comment line contains the number of vertices and the number of edges (undirected edge count, so the actual directed edge count is 2x this).

## Graph Categories

### Small / Verifiable (hand-checkable BFS results)
| File | Type | V | E | Description |
|------|------|---|---|-------------|
| `tiny_line.edgelist` | Path/Line | 5 | 4 | Simple path graph |
| `tiny_cycle.edgelist` | Cycle | 6 | 6 | Simple cycle |
| `tiny_complete.edgelist` | Complete | 6 | 15 | K6 complete graph |
| `tiny_tree.edgelist` | Tree | 7 | 6 | Binary tree (depth 2) |
| `tiny_bipartite.edgelist` | Bipartite | 8 | 8 | Complete bipartite K4,4 subset |
| `tiny_disconnected.edgelist` | Disconnected | 8 | 5 | Two disconnected components |
| `tiny_star.edgelist` | Star | 8 | 7 | Star graph (hub + spokes) |
| `tiny_grid.edgelist` | Grid | 9 | 12 | 3x3 grid |

### Medium (structural variety, still debuggable)
| File | Type | V | E | Description |
|------|------|---|---|-------------|
| `medium_sparse.edgelist` | Sparse random | 1000 | ~3000 | Avg degree ~6 |
| `medium_dense.edgelist` | Dense random | 1000 | ~50000 | Avg degree ~100 |
| `medium_chain.edgelist` | Path | 1000 | 999 | Worst-case BFS depth |

### Large (performance benchmarking)
| File | Type | V | E | Description |
|------|------|---|---|-------------|
| `large_sparse.edgelist` | Sparse random | 100000 | ~500000 | Avg degree ~10 |
| `large_dense.edgelist` | Dense random | 100000 | ~5000000 | Avg degree ~100 |
| `large_powerlaw.edgelist` | Power-law (RMAT) | 100000 | ~1000000 | Skewed degree distribution |

## Generation

Small graphs are hand-written. Medium and large graphs are generated by `generate_graphs.py`.

```bash
cd test-graphs
python3 generate_graphs.py
```
